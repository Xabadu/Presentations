React Nativo

+

La idea es chocar con el interprete de JS de los celulares, con Javascript mismo. Usar Javascript desde el entorno de desarrollo, con un servidor de Node. Que el interprete de JS del celular, lo transforme a componentes nativos (Java, u ObjectiveC). Y nos muestre la aplicación :)
